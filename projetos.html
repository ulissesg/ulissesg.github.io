Software para controle de rebanho na atividade leiteira

Projeto desenvolvido na UFPR - Setor Palotina, com a coordenação do Professor Marcos Antonio Schreiner. O projeto visa pesquisar sobre softwares de gerenciamento de rebanho, com licença livre disponíveis, suas funcionalidades e requisitos, e a partir da pesquisa desenvolver um software livre para essa necessidade.

Dispositivo para monitoramento de atividade de ovinos.

Projeto desenvolvido na UFPR - Setor Palotina, com a coordenação do Professor Marcos Antonio Schreiner. O projeto consiste em desenvolver um dispositivo capaz de monitorar os ovinos e determinar se os animais estão deitados, em pé, rumando, comendo ou bebendo água. Sendo usado uma coleira que contém um ARDUINO e alguns sensores, como o Altímetro Barômetro, leitor de RFID e sensor de vibração, para tornar o projeto realidade.
